How to Play

  1. Create a session Flip the first card & tap "Create Session"
  2. Scan the QR code WhatsApp → Linked Devices → Link
  3. Activate in a group Type !play in any group chat
  4. Start playing
    !poker start 50/100 !poker join 5000 !poker deal
